# 🔐 Password Generator Web App

A modern Password Generator built using **HTML5, CSS3, and JavaScript**. This application helps users generate secure passwords with customizable options and provides a clean Glassmorphism-based user interface.

## 🚀 Features

* Generate secure random passwords
* Password length slider (4–32 characters)
* Include/Exclude:

  * Uppercase Letters
  * Lowercase Letters
  * Numbers
  * Special Characters
* Password Strength Indicator
* Copy Password to Clipboard
* Dark/Light Mode Toggle
* Glassmorphism UI Design
* Toast Notifications
* Accessibility-Friendly Interface
* Smooth Animations and Hover Effects

##  Tech Stack

* HTML5
* CSS3
* JavaScript

## 📖 Usage

1. Select the desired password length.
2. Choose character types (uppercase, lowercase, numbers, symbols).
3. Click **Generate Password**.
4. Check the password strength indicator.
5. Click **Copy** to save the password to your clipboard.
